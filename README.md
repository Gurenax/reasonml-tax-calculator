# Calculate your Aussie Salary using ReasonML

Hello! This project allows you to calculate your Aussie Salary.
Last Updated: 2018 :)

# Build
```
yarn build
```

# Build + Watch
```
yarn start
```

# Run
```
node src/App.bs.js
```

# Editor
If you use `vscode`, Press `Windows + Shift + B` it will build automatically
